amabri卸载

发表于 2018-02-05

1.删除hdp.repo和hdp-util.repo

cd /etc/yum.repos.d/
rm -rf hdp
rm -rf HDP
rm -rf ambari*

2.删除安装包

用yum list installed | grep HDP来检查安装的ambari的包

5.重置数据库，删除ambari包

rm -rf /var/lib/ambari
rm -rf /var/log/ambari
